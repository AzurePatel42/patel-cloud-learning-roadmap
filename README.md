# Cloud Learning Roadmap — 5 Cloud‑Native Backend Projects
A complete, hands‑on journey to become a **Cloud Backend Engineer**, built with FastAPI, Docker, PostgreSQL, Azure, and Terraform.

---

## ⭐ Overview
This repository contains **five real backend projects**, each designed to build production‑level skills in:

- FastAPI  
- Docker & Docker Compose  
- PostgreSQL  
- Cloud deployment (Azure)  
- Infrastructure as Code (Terraform)  
- Git & GitHub workflow  
- CI/CD fundamentals  

This roadmap is structured to take you from **beginner → job‑ready backend engineer** through practical, cloud‑native development.

---

## ⭐ Your 5‑Project Portfolio
Each project is a standalone backend service with its own documentation, Docker setup, and API.

### **1️⃣ Cloud Tasks API**  
A CRUD Task Management API using FastAPI, PostgreSQL, and Docker.  
**Status:** Completed  
**Folder:** `/project-1-cloud-tasks-api`

### **2️⃣ File Storage API**  
Upload, download, and manage files using FastAPI + Azure Blob Storage.  
**Status:** Upcoming  
**Folder:** `/project-2-file-storage-api`

### **3️⃣ Auth Service (JWT + OAuth2)**  
User authentication, refresh tokens, hashing, and secure sessions.  
**Status:** Upcoming  
**Folder:** `/project-3-auth-service`

### **4️⃣ Analytics & Logging API**  
Event tracking, logging, and analytics pipeline with PostgreSQL + background workers.  
**Status:** Upcoming  
**Folder:** `/project-4-analytics-api`

### **5️⃣ AI‑Powered Lead Generation Platform**  
Full backend service integrating AI, tasks, users, and lead scoring.  
**Status:** Upcoming  
**Folder:** `/project-5-ai-leadgen-platform`
